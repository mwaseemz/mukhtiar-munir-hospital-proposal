## Technology Stack

- **Frontend:** Next.js 14, TypeScript, TailwindCSS
- **Backend:** Node.js, Express, Prisma
- **Database:** PostgreSQL (Supabase)
- **Cache:** Redis (Upstash)

## Features

✅ Patient Management with auto-generated MR numbers
✅ Authentication & Role-Based Access Control
✅ Digital Signatures
✅ Consent Forms (Urdu support)
✅ Audit Trail
✅ Redis Caching
